# Investment Framework

This framework is used by the Investment Agent to produce disciplined investment decision support.

## 1. Investment Question

Define the question before analysis:

```text
Are we evaluating:
- A new position?
- Adding to an existing position?
- Trimming or exiting?
- Watchlist only?
- Sector or macro exposure?
```

## 2. Required Inputs

- User objective and time horizon
- Portfolio policy
- Current portfolio context, if available
- Company fundamentals
- Industry structure
- Valuation
- Catalysts
- Risks
- Alternative opportunities

## 3. Core Thesis

A thesis should answer:

- Why this asset?
- Why now or why wait?
- What must be true for the investment to work?
- What could break the thesis?
- How will we know we are wrong?

## 4. Evaluation Areas

| Area | Questions |
| --- | --- |
| Business Quality | Is the company durable, differentiated, and profitable? |
| Growth | Is growth structural, cyclical, or temporary? |
| Valuation | Is the expected return attractive versus risk? |
| Catalysts | What can unlock upside? |
| Risks | What can impair capital or compress valuation? |
| Portfolio Fit | Does it improve or over-concentrate the portfolio? |
| Timing | Is current price attractive, fair, or stretched? |

## 5. Output Categories

Use one of these action categories:

- Strong candidate
- Candidate, wait for price
- Watchlist only
- Avoid
- Review existing position
- Trim / reduce risk

## 6. Position Sizing Logic

Position sizing must consider:

- User risk tolerance
- Maximum single-name exposure
- Volatility
- Drawdown risk
- Confidence level
- Correlation with existing holdings

## 7. Review Discipline

Every investment memo should include:

- Review date
- Tracking indicators
- Anti-thesis
- Exit conditions

## 8. Safety Rule

Investment Agent output is decision support only. Final investment decisions remain with the user.
